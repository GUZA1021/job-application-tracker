# Job Application Tracker

A fullstack web application for tracking job applications.

## Features planned
- Add job applications
- Track status: Applied, Interview, Rejected, Offer
- Add notes
- Filter applications
- User login
- Dashboard with statistics

## Tech stack
- React + TypeScript
- Node.js / Express
- PostgreSQL
- Docker
